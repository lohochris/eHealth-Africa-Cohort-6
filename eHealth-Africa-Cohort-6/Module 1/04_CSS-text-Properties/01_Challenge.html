Font Family Examples

Serif Font

This is a paragraph with the serif font family. Serif fonts are traditional and often used in print. Each letter has small decorative lines at the end of its strokes.

Sans-Serif Font

This is a paragraph with the sans-serif font family. Sans-serif fonts are modern and clean, without the decorative lines seen in serif fonts.

Monospace Font

This is a paragraph with the monospace font family. Monospace fonts make each character occupy the same width, which is ideal for coding and technical documents.

Cursive Font

This is a paragraph with the cursive font family. Cursive fonts imitate handwriting and can add a creative, personal touch to the text.

Fantasy Font

This is a paragraph with the fantasy font family. Fantasy fonts are highly decorative and used for creative, playful styles rather than readability.
